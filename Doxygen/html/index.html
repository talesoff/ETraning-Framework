시작하기

이 문서에서는 ETE.Engine을 소개합니다.

ETE.Engine은 이트레이닝시스템 공통플랫폼 코어 엔진 개발 과제의 산출물로서 시뮬레이션 코어 프레임워크입니다.

.NET Framework를 기반으로 개발 되었고 .NET Framework에 호환되는 실행환경에서 ETE.Engine을 사용할 수 있습니다.

이 문서가 필요한 사람

이 문서는 과제 참여자/참여사에서 코어 엔진을 개발하는 사람을 대상으로 작성되었습니다.

새로운 코어 엔진 개발 또는 기존 엔진의 코어 엔진을 시뮬레이션 엔진에 연동하여 최종 시뮬레이터 환경을 만들 수 있습니다.

3rd party 개발자 또는 시뮬레이터의 사용자는 설명서를 참고하세요.

개발 환경

  • Visual Studio 2013 이상 (Windows)
  • .NET 4.5 이상 (Mono 4.0.0 이상)
  • C# 5.0 이하
  • 기타 .NET 호환 개발 환경

실행 환경

  • .NET Framework 4.5 이상 (Mono Runtime 4.0.0 이상)

  • VB 9, Java, Phython, Ruby, Eiffel, F#, Oxygene, ... (.NET Framwork와 Mono에서 지원하는 기타 언어)

주의

  • 프로젝트 구성

    ETE.Engine 바이너리(dll)는 실행하는 응용프로그램과 같은 경로에 포함하여 프로젝트를 구성하시기 바랍니다. 시뮬레이션에서 사용하는 상대 경로는 ETE.Engine 바이너리의 위치를 기준으로 설정되어 있습니다.
